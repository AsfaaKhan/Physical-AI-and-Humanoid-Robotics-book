# Glossary

This glossary provides definitions for key terms used throughout the Physical AI & Humanoid Robotics book.

## A

**Affordance**: The possibility of an action that an object or environment provides to an agent.

**Artificial General Intelligence (AGI)**: Hypothetical AI that can understand, learn, and apply knowledge across diverse domains at human level or beyond.

## B

**Bipedal locomotion**: Walking on two legs, a key capability for humanoid robots.

**Behavior tree**: A hierarchical structure used in robotics and AI to organize and execute complex behaviors.

## C

**Center of Mass (CoM)**: The point where the total mass of a body can be considered concentrated for the purpose of analyzing motion.

**Collision detection**: The computational problem of detecting when two or more bodies come into contact.

**Computer Vision**: A field of AI that trains computers to interpret and understand the visual world.

## D

**Deep Learning**: A subset of machine learning based on artificial neural networks with representation learning.

**Dynamic Balance**: Maintaining balance through active control rather than static stability.

## E

**Embodied AI**: AI systems that interact with the physical world through a physical body.

**End-effector**: The device at the end of a robotic arm, designed to interact with the environment.

## F

**Forward Kinematics**: The process of determining the position and orientation of the end-effector given the joint angles.

**Fused deposition modeling (FDM)**: A 3D printing technology that uses a thermoplastic filament.

## G

**Gazebo**: A robotics simulator that provides realistic physics simulation and convenient programmatic interfaces.

**Generalized coordinates**: A set of parameters that define the configuration of a mechanical system.

## H

**Human-Robot Interaction (HRI)**: The study of interactions between humans and robots.

**Hyperparameter**: A parameter whose value is set before the learning process begins.

## I

**Inverse Kinematics**: The process of determining joint angles required to achieve a desired end-effector position.

**Isaac Sim**: NVIDIA's high-fidelity simulation environment built on the Omniverse platform.

## J

**Joint Space**: The space defined by the joint angles of a robot.

**Jacobian Matrix**: A matrix that contains first-order partial derivatives of a vector-valued function.

## K

**Kinematics**: The branch of mechanics concerned with the motion of objects without reference to force.

## L

**LIDAR**: Light Detection and Ranging, a remote sensing method that uses light in the form of a pulsed laser.

**Locomotion**: The ability to move from one place to another.

## M

**Manipulation**: The ability to purposefully change the state of objects in the environment.

**Model Predictive Control (MPC)**: An advanced control method that uses a model of the system to predict future behavior.

## N

**Navigation**: The process of planning and executing a path from a start to a goal location.

**Neural Network**: A series of algorithms that endeavors to recognize underlying relationships in a set of data.

## O

**Omnidirectional**: Capable of moving in any direction.

**OpenVLA**: Open Vision-Language-Action framework for embodied AI.

## P

**Perception**: The process of acquiring, interpreting, selecting, and organizing sensory information.

**PID Controller**: A control loop mechanism employing feedback that is widely used in industrial control systems.

**Point Cloud**: A set of data points in space, typically defining the external surface of an object.

## R

**ROS (Robot Operating System)**: A flexible framework for writing robot software.

**ROS2**: The second generation of the Robot Operating System.

**Rigid Body**: An idealization of a solid body in which deformation is neglected.

## S

**SLAM (Simultaneous Localization and Mapping)**: The computational problem of constructing or updating a map of an unknown environment while simultaneously keeping track of an agent's location within it.

**Stability**: The ability of a system to return to its equilibrium state after a disturbance.

## T

**Task Planning**: The process of determining a sequence of actions to achieve a specified goal.

**Trajectory**: The path that an object follows through space as a function of time.

## U

**URDF (Unified Robot Description Format)**: An XML format used in ROS to describe robot models.

## V

**VLA (Vision-Language-Action)**: Systems that combine visual perception, language understanding, and physical action.

**Vision-Language Model (VLM)**: AI models that can understand both visual and linguistic information.

## W

**Whole-Body Control**: A control approach that considers the entire robot as a single system for coordinated movement.
